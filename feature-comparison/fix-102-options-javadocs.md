# Fix 102

101 muss abgeschlossen sein.


Die enums Option, SchedulerOption, CallbackOption und SchedulerCallbackOption repräsentieren die meisten Konfigurationsmöglichkeiten des FullCalendars. Einige haben ausführliche JavaDocs, andere
hingegen nur einen Link auf die FC Docs Seite. 

Ich möchte, dass die Javadocs jedes enums hier dem User einen Überblick über die Option geben, sowie was hier gesetzt werden kann, was der
default ist usw. Wenn eine Option auch JS Funktionen erlaubt, soll auf die passende CallbackOption diesbezüglich verwiesen werden (und vice versa).
